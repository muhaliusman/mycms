# Cwcms

## Requirements

- Min Laravel 5.4

## Installation

Setup your environtment in your .env file.

Then install via Composer

``` bash
$ composer require cerberusworks/cwcms
```

After finish instalation process, next run install command

``` bash
$ php artisan cwcms:install --with-dummy
```

Finish
